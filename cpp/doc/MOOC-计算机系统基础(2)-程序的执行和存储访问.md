# 计算机系统基础（三）：异常、中断与输入输出

## week7 I/O方式和内核空间I/O软件

**三种基本的I/O方式**

### 1.程序直接控制
`无条件传送`：比如定时进行传送
`条件传送`：轮询polling

### 2.中断I/O模式
中断请求，中断服务，中断服务程序

### 3.DMA方式
针对高速外设,eg,磁盘

**请求-响应模式**
磁盘准备好就向DMA控制器发送`DMA请求`
                  |
                  |
DMA控制器接接收到请求后，向CPU申请总线控制权
                  |
                  |
CPU让出总线控制权去干其他事情，DMA通过总线在内存和磁盘之间传输数据 *dma拥有比cpu更高的总线使用权*
                  |
                  |
DMA传送结束后，通过中断告诉CPU
